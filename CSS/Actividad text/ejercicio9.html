- ¿Que hace?

- Nada.

- Oye, ¿que es peor: la ignorancia o el desinteres?

- Ni lo se ni me importa.

¿Que diferencia hay entre uno que se cae de un octavo piso y uno que se cae desde un primer piso?

El que se cae desde el octavo piso: Ay, ay, ay, ay, ay... ¡Poooof!

El que se cae desde el primer piso: ¡Pooof! Ay, ay, ay, ay, ay...
